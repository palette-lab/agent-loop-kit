# Positioning

Agent Loop Kit is not only a prompt catalog. It adds:

| Capability | Plain prompt gallery | Agent Loop Kit |
| --- | --- | --- |
| Copyable prompts | yes | yes |
| Machine-readable loop schema | no | yes |
| Validation in CI | no | yes |
| CLI search and export | no | yes |
| `AGENTS.md` export | no | yes |
| MCP prompt/resource server | no | yes |
| Evidence contracts | inconsistent | required |
| Human approval gates | inconsistent | required |
| Contribution quality bar | often informal | documented and tested |

The project should stay small enough to be useful from `npx`, but structured enough for agent platforms and open-source maintainers.
